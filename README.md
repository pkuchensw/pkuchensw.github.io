# Siwei Chen Personal Homepage

This is a static academic homepage for GitHub Pages.

## Files

```text
index.html
style.css
assets/avatar.jpg
```

## Deploy to GitHub Pages

1. Create a GitHub repository named:

```text
pkuchensw.github.io
```

2. Upload all files in this folder to the repository root.

3. Open:

```text
Settings → Pages
```

4. Select:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

5. Visit:

```text
https://pkuchensw.github.io/
```

## Replace the avatar

Replace `assets/avatar.jpg` with your own photo. Keep the filename as `avatar.jpg`, or update the image path in `index.html`.
